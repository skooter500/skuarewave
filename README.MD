Proudly created with Godot!

--------------------------------------------------
- Play by waving your hands inside the skuares
- Experiment with instrument octave root-note and mode
- Pinch in a skuare to program the note in the step
- Hit start to activate the sequencers
- Change the direction with the controls
- Add reverb and delay effects
- Bring in the mic
